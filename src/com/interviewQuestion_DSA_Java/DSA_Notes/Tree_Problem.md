# 1) Tree Traversal Pattern (MOST BASIC + VERY IMPORTANT)

### Recognition:

* Tree visit karna hai
* Root / Left / Right order matter karta hai

---

## Top Interview Questions

### DFS Traversals

* Preorder Traversal of Binary Tree ⭐⭐⭐⭐⭐
  (Root → Left → Right)

* Inorder Traversal of Binary Tree ⭐⭐⭐⭐⭐
  (Left → Root → Right)

* Postorder Traversal of Binary Tree ⭐⭐⭐⭐⭐
  (Left → Right → Root)

---

### BFS Traversal

* Level Order Traversal of Binary Tree ⭐⭐⭐⭐⭐
* Zigzag Level Order Traversal ⭐⭐⭐⭐⭐
* Reverse Level Order Traversal ⭐⭐⭐⭐

Example:

```java
DFS → Recursion / Stack
BFS → Queue
```
